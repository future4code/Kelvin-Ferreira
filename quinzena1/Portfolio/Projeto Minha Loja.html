PROJETO MINHA LOJA

Este é meu segundo projeto até o momento, faz parte de um exercício do curso da Labenu. Aqui eu criei um "website" de uma loja fictícia de jogos de video-game. O site possui seis páginas no total, cada uma se refere a uma área e função específica da loja: Temos a página inicial, o estoque, a página de vendas, o histórico de vendas, uma área para adicionar novos produtos e uma última com a lista de funcionários. E se comparado ao primeiro projeto, neste eu já consegui montar algo muito mais organizado e ordenado, isso devido ao desenvolvimento que tive com o HTML e, principalmente, com o CSS. O site no entanto, apenas utiliza desses dois códigos, diferentemente do primeiro onde havia um pouco de JavaScript.
